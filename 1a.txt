1
Install packages
sudo apt install openvpn easy-rsa -y
openvpn = VPN software · easy-rsa = certificate management tool
2
Create PKI directory & Init
make-cadir ~/openvpn-ca → cd ~/openvpn-ca → ./easyrsa init-pki
Creates the folder structure to hold all keys and certs
3
Build Certificate Authority
./easyrsa build-ca — Enter name e.g. "MyVPN-CA". This is the root trust anchor.
4
Generate Server Certificate
./easyrsa gen-req server nopass → ./easyrsa sign-req server server → type yes
Creates server identity that clients verify
5
Generate DH params + TLS key
./easyrsa gen-dh → openvpn --genkey secret ta.key
DH = key exchange security · ta.key = extra HMAC protection
6
Copy all certs to OpenVPN dir
sudo cp pki/ca.crt pki/private/server.key pki/issued/server.crt pki/dh.pem ta.key /etc/openvpn/
7
Create server.conf
sudo nano /etc/openvpn/server.conf — Key lines: server 10.8.0.0 255.255.255.0 (VPN subnet) · port 1194 · proto udp · cipher AES-256-CBC
8
Enable IP Forwarding
sudo nano /etc/sysctl.conf → add net.ipv4.ip_forward = 1 → sudo sysctl -p
Allows VPN server to route packets between VPN clients and the internet
9
Start & Verify
sudo systemctl start openvpn@server → sudo systemctl enable openvpn@server → ip a — look for tun0 with IP 10.8.0.1
